# 实习能量站：业务部实习生成长导航 AI 看板

Demo 使用 Streamlit 搭建，不接入真实 AI API，不需要 API Key；系统通过规则引擎模拟 AI 决策逻辑，基于任务完成率、岗位技能、导师反馈、学习主动性、协作评分等结构化数据，自动生成 HR 跟进建议、导师反馈和 HR 周报。

正式落地时，可接入腾讯混元 / 元宝，将结构化数据、导师反馈文本和风险原因作为 Prompt 输入，由大模型生成更自然、更个性化的成长建议。当前版本优先保证公网部署稳定、免费、无需登录即可演示。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 公网部署

推荐使用 Streamlit Community Cloud：

1. 将提交文件上传到 GitHub 仓库。
2. 打开 https://share.streamlit.io/。
3. 选择仓库、分支和入口文件 `app.py`。
4. 点击 Deploy。
5. 获得形如 `https://your-app-name.streamlit.app` 的公网链接。

不需要配置任何 Secrets 或 API Key。

## 建议上传到 GitHub 的文件

## Demo 页面

- 首页总览：展示整体适岗指数、风险人数、高潜人数、AI 周报，并支持下载 HR 周报。
- 实习生列表：支持筛选、排序和生成 HR 跟进建议。
- 个人详情：展示 30/60/90 天成长路径、能力雷达、导师反馈和 AI 建议。
- 导师工作台：展示待办事项、反馈生成器和标准带教模板。
- 成长趋势：展示周度适岗指数趋势、岗位趋势、满意度与适岗指数关系、风险原因拆解。
- HR 分析看板：展示岗位对比、风险分布、高潜人才、风险人才、风险原因和转正潜力。
- 右上角语言切换：支持中文 / English 两种界面语言。
- AI 模式说明：支持规则引擎模式，以及“大模型模式（未来扩展，不调用 API）”说明。

## 上传自己的数据

页面左侧支持上传 CSV 文件。字段结构请参考 `interns.csv`，也可以在页面中点击“下载数据模板”。上传后，系统会自动重新计算适岗指数、风险等级和转正潜力，并刷新所有看板与 AI 建议。

## 指标说明

实习生适岗指数 = 30% 任务完成率 + 25% 岗位技能评分 + 20% 导师反馈评分 + 15% 学习主动性 + 10% 沟通协作评分。

风险等级：

- 85-100：高潜
- 70-84：健康成长
- 55-69：需要关注
- 55 以下：高风险
